Electromagnetic Walking Hexapod — Navy Hull Inspection
MI Technical Solutions / R&D
java · maven · raspberry pi 5 · arduino · bno055 imu
A six-legged walking platform that adheres to ferromagnetic naval hulls using electromagnet-tipped feet. A Java controller running on the Raspberry Pi 5 handles gait sequencing and leg coordination, dispatching joint commands over serial to Arduino Mega/Nano boards that drive the servo array through a PCA9685 16-channel PWM controller. A BNO055 IMU provides real-time orientation feedback over I2C for balance, while an Arducam IMX519 streams the live inspection feed.
  • Multi-module Maven codebase (shared / controller / robot) running on a Raspberry Pi 5 with Arduino Mega/Nano over serial.
  • Servo control via PCA9685, orientation sensing with a BNO055 IMU, and live camera streaming from an Arducam IMX519.
  • Built to walk along naval ship hulls and detect structural and safety issues during inspection.
AI-Powered DevOps Automation
Microsoft AI Dev Days 2026 — Hackathon
azure ai foundry · c#/.net 8 · gpt-4o · signalr
An automated DevOps pipeline that chains six specialized GPT-4o agents in sequence, each constrained by its own system prompt to emit deterministic, schema-validated JSON that the next stage consumes. Static analysis results are normalized to the SARIF standard, a human-in-the-loop approval gate blocks promotion until a reviewer signs off, and a SignalR WebSocket layer pushes live pipeline state to a real-time dashboard.
  • Designed an end-to-end six-agent sequential pipeline where each agent calls GPT-4o with a specialized prompt and emits structured JSON.
  • Implemented a human-in-the-loop approval gate and SARIF-compliant static analysis findings.
  • Integrated a real-time SignalR dashboard for live pipeline visibility.
Misinformation Detection Web Application
Full-stack team project
node.js · express · react · hugging face · mysql
A full-stack credibility analysis service that ingests a news URL, scrapes and extracts the article body, and routes it through a Hugging Face language model that scores it across domain reputation, political bias, citation quality, and content integrity. The weighted sub-scores feed a scoring algorithm that returns a normalized 0–100 credibility rating, with analysis history persisted to a MySQL store behind JWT-authenticated user sessions.
  • Integrated a Hugging Face LLM to evaluate news articles in real time across domain reputation, bias, citations, and content quality.
  • Fed structured JSON into a scoring algorithm that produces a 0–100 credibility score.
  • Added web scraping, article extraction, and a MySQL-backed history system with JWT authentication.
