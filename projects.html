AI-assisted Navy maintenance reporting
Founding project

A React application that converts raw field evidence — photos and inspection notes — into formal, submission-ready Navy maintenance-availability reports. Engineered LLM prompts return strict structured JSON parsed against a typed report schema, then a human-in-the-loop pipeline (AI draft → edit → section-level QA verification → sign-off → export) keeps a reviewer in control. Includes per-line evidence traceability, an offline deterministic fallback when the API is unavailable, a submission-ready export mapped to Navy database fields, and a CUI / data-residency-aware security posture.
