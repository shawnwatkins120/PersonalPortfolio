about

I'm a Computer Science student at Old Dominion University (B.S., expected August 2026) and a software engineer based in the Virginia Beach / Norfolk area. My focus is building reliable, well-structured software across the full stack — from web applications and AI-driven services down to low-level embedded firmware — with an emphasis on clean architecture and maintainable, testable code.

Right now I lead software development for an R&D team at MI Technical Solutions, building a walking robot for naval hull inspection. The work spans the entire system: Java application logic organized as a multi-module Maven project on a Raspberry Pi 5, motor-control and serial-communication firmware on Arduino microcontrollers, sensor integration over I2C, and real-time camera streaming. It's taught me to write software that holds up against real hardware and real-world constraints.

My background also includes network infrastructure and a Governance, Risk, and Compliance application built during an IT internship with the City of Chesapeake, plus AI and full-stack projects ranging from a six-agent GPT-4o automation pipeline to a misinformation-detection web app. I'm comfortable in Agile teams working with Git-based workflows and CI/CD, and I work primarily in Java, Python, and C#/.NET.

Outside of engineering, I have conversational Japanese and personal ties to Japan, and a strong interest in defense and government technology. I'm always looking for the next hard problem to build through.
