Featured Project

Keelson

AI-assisted Navy maintenance reporting
Founding project

A React application that turns raw field evidence — inspection photos and notes — into formal, submission-ready Navy maintenance-availability reports. Engineered LLM prompts return strict structured JSON parsed against a typed report schema, while a human-in-the-loop pipeline (draft → edit → section-level QA verification → sign-off → export) keeps a reviewer in control at every step. Each line of a report stays traceable to the evidence behind it, a deterministic offline fallback keeps the tool working when the API is unavailable, and the final export maps cleanly to Navy database fields under a CUI-aware data posture.
